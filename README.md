## 📋 Common Troubleshooting

If you encounter issues during installation or setup, try these solutions.

### Installation Errors
If the installer closes unexpectedly, check your available disk space. Ensure that your firewall or antivirus software does not block the installer. You may need to disable the antivirus temporarily while you run the setup.

### Performance Problems
If the application runs slowly, update your graphics card drivers. Visit the website of your graphics card manufacturer to download the latest driver software. Additionally, ensure that your computer uses the dedicated GPU for ArchiCAD rather than the integrated graphics chip.

### Missing Libraries
If the software cannot find certain building components, navigate to the library manager. Use the "Add" button to point the software toward your library folder. You can download extra library assets from the official library portal if necessary.

## 📐 Using the Interface

ArchiCAD 2026 uses a ribbon-based interface. The top menu holds the primary tools for drawing walls, slabs, windows, and doors. The bottom status bar shows your current scale and zoom level.

- Use the "Toolbox" on the left to select construction elements.
- Use the "Navigator" on the right to switch between floor plans, sections, and 3D views.
- Use the "Properties" panel to adjust the height, width, and materials of selected elements.

To start a new project:

3. Select a template based on your region or project type.
